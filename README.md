# Nebula: Multi-Agent LLM Orchestration System 🚀

Курсова робота, присвячена розробці та дослідженню архітектури мультиагентної оркестрації великих мовних моделей (LLM) у стохастичному контекстному просторі. Проєкт реалізовано у вигляді працездатного MVP-мікросервісу з аналітичним веб-інтерфейсом управління.

## 📌 Основна проблематика
Звичайні LLM-рішення часто стикаються з проблемою "чорної скриньки", галюцинаціями та відсутністю гнучкого контролю над контекстом. **Nebula** вирішує ці виклики завдяки чіткому розділенню обов'язків між спеціалізованими агентами, використанню суворого когнітивного циклу міркувань та автоматичній оцінці якості відповідей.

## 🛠️ Ключові компоненти та метрики MVP
- **Семантична маршрутизація:** Розроблено таксономію із **62 унікальних інтенцій** (намірів користувача) для точного визначення контексту запиту.
- **Рольова мультиагентність:** Спроєктовано **7 спеціалізованих профілів агентів** із власними інструкціями (System Prompts).
- **Фактологічна база знань (RAG):** Векторне сховище на базі **ChromaDB**, що містить **175 векторизованих чанків** еталонних знань доменної області.
- **Когнітивний контур ReAct (Reasoning + Acting):** Змушує модель ітераційно логувати свої думки (`Thought`), виконувати дії пошуку (`Action`) та аналізувати результати (`Observation`).
- **Автоматичний офлайн-скоринг (Proxy Score):** Інтегральна математична метрика якості сесії (діапазон 0-1), що враховує релевантність контексту, точність інтенції та відсутність галюцинацій.

## 💻 Структура веб-інтерфейсу (Панелі керування)
1. **Main Screen:** Інференс у реальному часі (середня затримка — **450 мс**) з демонстрацією відповідей оркестратора.
2. **Workflows:** Автоматизовані сценарії (відповіді на ліди, тікети підтримки,Slack-звіти).
3. **Logs (Observability):** Повне трасування когнітивних ланцюжків ReAct для аудиту та збору даних.
4. **Settings:** Динамічне керування промптами, температурою, лімітами токенів та вагами `proxy_score` без перевантаження серверного коду.

## 🏗️ Технологічний стек (Tech Stack)
- **Backend:** Python, FastAPI, Uvicorn, Pydantic (валідація схем даних)
- **AI/LLMOps:** ChromaDB (Vector DB), ReAct Pattern, RAG Architecture
- **Frontend:** HTML5, CSS3, JavaScript (асинхронна взаємодія через REST API)
